Proyecto Alvaro Toth

Como primera medida decidí usar solo css para los estilos porque me parece que para este proyecto no es necesario usar algún framework como bootstrap ya que estos son simples. Luego para consumir el array de productos primero lo había puesto en una carpeta aparte de js donde usaba la función getData para enviar los productos y luego la usaba en los componentes, pero me resulto más simple ponerla directamente en los componentes donde iba a usarlo. 
La selección de productos vino de cursos anteriores como el de desarrollo web y js donde realice una página web sobre la empresa familiar.
